# Prerequisites

- Python 3.x
- nltk - can be installed by: 

    ```bash
    pip install nltk
    ```

- nltk data - invoke the python interpreter and run the following code block:

    ```python
    import nltk
    nltk.download('stopwords')
    nltk.download('punkt')
    ```

# Running the model

To run the model, just run:

```bash
python main.py
```
